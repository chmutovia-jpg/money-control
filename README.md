# Money Control

Money Control — локальный трекер личных финансов в стиле premium fintech. Приложение помогает вести доходы и расходы, контролировать подписки, долги, цели накоплений, бюджеты, счета и прогноз денег до конца месяца.

Демо: https://chmutovia-jpg.github.io/money-control/

## Возможности

- Dashboard с дневным лимитом, балансом, прогнозом и финансовой температурой.
- Счета и кошельки: карта, наличные, накопления, кредитный, crypto и другое.
- Доходы, расходы, регулярные операции и быстрый ввод вроде `кофе 250`.
- Подписки, долги, цели накоплений, бюджеты по категориям и календарь платежей.
- Cashflow-прогноз, стресс-тест покупки, режим экономии и локальные Smart Alerts.
- Аналитика, сравнение с прошлым месяцем и Month Wrapped.
- Локальный профиль, PIN-lock, скрытие сумм, выбор темы и аватар.
- Импорт/экспорт JSON, экспорт CSV и резервная копия.
- PWA: приложение можно добавить на экран телефона.

## Как запустить

```bash
npm install
npm run dev
```

Открой локальный адрес, который покажет Vite, обычно `http://localhost:5173/`.

## Как собрать

```bash
npm run build
```

Готовая сборка появится в папке `dist`.

## Как запустить тесты

```bash
npm test
```

## GitHub Pages

Проект настроен под GitHub Pages с относительным base `./`. Деплой можно делать через GitHub Actions из `.github/workflows/deploy.yml`.

## Ограничения localStorage

Money Control не использует backend и облако. Все данные хранятся только в текущем браузере на текущем устройстве. Если очистить данные браузера или открыть приложение на другом телефоне, данные не появятся автоматически. Для переноса используй экспорт JSON и импорт JSON.
